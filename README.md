安装typescript：

```
npm install -g typescript
```

单独执行文件：
```
tsc greeter.ts
```

**NOTE:** gulpfile.js为空文件，把需要执行的gulp内容拷贝到gulpfile.js里。

项目执行命令：
```
gulp
```
